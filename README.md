# Personalized-Weather-App

## Overview

This application allows you to register/login and view the current weather, and the 5-day weather forecast based on your specified zip-code. You also have the ability to switch between light and dark mode, and update your zipcode.

## Technologies used

✅ Rest API's  
✅ Local Storage  
✅ JavaScript  
✅ HTML  
✅ CSS
